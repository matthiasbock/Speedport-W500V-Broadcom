Telefonnummern gehend

Hier können Sie festlegen mit welcher Telefonnummer eine gehende Verbindung erfolgen soll.

Sie haben die Möglichkeit eine alternative Telefonnummer zu verwenden. Diese kann automatisch und manuell benutzt werden.

Telefonnummer gehend

Hier können Sie eine Prioritäts-Telefonnummer auswählen, die für gehende Verbindungen verwendet werden soll.


Notrufe über die Nummern 110 und 112 werden aus Gründen der Sicherheit grundsätzlich über das Festnetz abgesetzt. Nur in Fällen, in denen das nicht mög- lich ist, wird eine andere Ver- bindungsmöglichkeit gesucht.

Alternative Telefonnummer gehend

Die alternative Telefonnummer wird automatisch verwendet, falls die Prioritäts-Telefonnummer belegt oder nicht verfügbar ist.

Aktivieren Sie hierzu das Kontrollkästchen.

Alternative Telefonnummer gehend

Die alternative Telefonnummer kann durch Vorwahl von 2# gezielt für gehende Verbindungen verwendet werden.

Aktivieren Sie hierzu das Kontrollkästchen.

Telefonnummern unterdrücken

Die Telefonnummer zum Angerufenen kann unterdrückt werden.

Aktivieren Sie hierzu das Kontrollkästchen.

Alternative Telefonnummer gehend nicht verfügbar

Diese Einstellung kann erst vorgenommen werden, wenn sowohl eine Telefonnummer von einem Festnetzanschluss als auch eine Telefonnummer für die DSL Telefonie zur Verfügung steht.
