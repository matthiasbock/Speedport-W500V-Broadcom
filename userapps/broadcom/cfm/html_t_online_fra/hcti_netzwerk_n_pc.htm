PC ...

Sie können diesem PC Anwendungsregeln zuordnen. Wenn Sie Dateien oder Web-Dienste, die sich auf diesem PC Ihres lokalen Netzwerkes befinden, anderen Internet-Benutzern anbieten wollen, richten Sie diesen PC als Server ein. Die NAT-Funktion des Routers lässt nicht zu, dass von „außen“ auf Rechner des lokalen Netzwerks zugegriffen wird. Um Dienste von lokalen Rechnern im Internet zur Verfügung zu stellen, müssen Sie Anwendungsregeln einrichten. Dadurch übernimmt der Router nach außen die Rolle des Servers. Er empfängt die Anforderungen ferner Benutzer unter seiner öffentlichen IP-Adresse und leitet diese automatisch auf lokale PCs um. Die privaten IP-Adressen der Server im lokalen Netzwerk bleiben geschützt.

Weitere Informationen erhalten Sie, wenn Sie den Mauszeiger über einen Menüpunkt bewegen.
