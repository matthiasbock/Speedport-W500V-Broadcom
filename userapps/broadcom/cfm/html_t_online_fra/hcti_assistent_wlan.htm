Assistent / Sicherheitseinstellungen für WLAN Clients
Weitere Vorgehensweise
Verwenden Sie nun den Datenträger, auf den Sie die Sicherheits-
einstellungen kopiert haben, damit Sie die Einstellungen bei den
WLAN Clients vornehmen können.

Somit ist sicher gestellt, dass die Sicherheitseinstellungen im
Router exakt mit den Einstellungen beim WLAN Client übereinstimmen.

Stellen Sie vorher sicher, dass der WLAN Manager für den Client PC
installiert ist.

Wenn Sie den WLAN Manager öffnen, können Sie unter 'Import'
die Daten von Ihrem Datenträger importieren.
