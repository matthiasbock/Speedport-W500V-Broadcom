Konfigurationsdaten

Hier können Sie die Konfiguration Ihres Routers laden, sichern oder zurücksetzen.

Außerdem können Sie ein Firmwareupdate des Routers durchführen.

Weitere Informationen erhalten Sie, wenn Sie den Mauszeiger über einen Menüpunkt bewegen.

Konfigurationsdaten

Konfiguration laden
Eine bereits gesicherte Konfiguration können Sie wieder in Ihren Router zurück laden.

Konfigurationsdaten

Konfiguration sichern
Verwenden Sie 'Sichern', um die aktuelle Konfiguration in der 'config.bin' benannten Datei auf Ihrem Rechner zu sichern.

Konfigurationsdaten

Werkseinstellung
Hier können Sie Ihren Router wieder auf die Werkseinstellung zurücksetzen.

Sicherheitseinstellungen

Importieren
Bereits gespeicherte Sicherheitseinstellungen können Sie wieder in Ihren Router zurück laden.

Sicherheitseinstellungen

Speichern
Verwenden Sie 'Speichern', um die aktuellen Sicherheitsein- stellungen auf einem Daten- träger (USB-Stick oder Diskette) zu speichern. Somit können Sie die Sicherheitseinstellungen für die Konfiguration von anderen WLAN Clients verwenden.

Verwenden Sie nun den Daten- träger, auf den Sie die Sicher- heitseinstellungen kopiert haben, um die Einstellungen bei den WLAN Clients vornehmen zu können. Somit ist sicher gestellt, dass die Sicherheitseinstellun- gen im Router exakt mit den Einstellungen beim WLAN Client übereinstimmen.

Stellen Sie vorher sicher, dass der WLAN Manager für den Client PC installiert ist. Wenn Sie den WLAN Manager öffnen, können Sie unter 'Sicherheits- einstellungen' die Daten von Ihrem Datenträger importieren.

Firmware

Firmwareupdate
Hier können Sie die Firmware ihres Routers aktualisieren.

Die aktuelle Firmware wird Ihnen auf den Internetseiten der Deutschen Telekom AG zum Download zur Verfügung gestellt.

Bevor Sie die Firmware aktualisieren, laden Sie diese bitte zuerst auf Ihre Festplatte herunter.
