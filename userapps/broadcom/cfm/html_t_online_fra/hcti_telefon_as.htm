Anschluss-Einstellungen

Passen Sie den Anschluss Ihren persönlichen Erfordernissen an. Die Einstellung können Sie ändern, klicken Sie auf den gewünschten Menüpunkt.

Weitere Informationen erhalten Sie, wenn Sie den Mauszeiger über einen Menüpunkt bewegen.

Anschluss-Einstellungen

Rufnummern gehend
Sie können bestimmen mit welcher Rufnummer abgehende Verbindungen aufgebaut werden. Außerdem können Sie festlegen, ob die Rufnummer zur Gegenstelle übermittelt oder unterdrückt werden soll.

Anschluss-Einstellungen

Anzeige der Rufnummer (CLIP)
CLIP ist die Abkürzung für:
Calling
Line
Identification
Presentation.

Anzeige der Rufnummer des Anrufenden. Die Unterstützung durch das Endgerät ist notwendig.

Anklopfen
Bestimmen Sie, ob Sie während eines Gesprächs per Aufmerk- samkeitston mitgeteilt bekommen möchten, dass noch jemand anruft.

Rufnummern gehend nicht verfügbar

Es ist keine Internetrufnummer eingerichtet oder der analoger Telefonanschluss ausgeschaltet. Richten Sie eine Internetrufnummer ein und schalten Sie den analoger Telefonanschluss ein. Rufen Sie dann diesen Menüpunkt auf.

Rufnummern gehend nicht verfügbar

Es ist keine Internetrufnummer aktiviert oder keine T-ISDN Rufnummer eingetragen. Aktivieren Sie eine oder mehrere Internetrufnummern und/oder tragen Sie eine oder mehrere T-ISDN Rufnummern ein und rufen Sie dann diesen Menüpunkt auf.
