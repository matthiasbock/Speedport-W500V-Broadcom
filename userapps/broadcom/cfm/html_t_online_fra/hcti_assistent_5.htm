Bitte konfigurieren Sie zu Ihrer Sicherheit die Verschlüsselung !

Durch Verschlüsselung wird die Übertragung der Daten in Ihrem drahtlosen Netzwerk vor unbe- fugtem Abhören geschützt.

Ein Verschlüsselungsalgorith- mus chiffriert Ihre Daten und macht sie somit unleserlich. Bei Kenntnis des dabei verwendeten Schlüssels können die Daten auf der Empfangsseite wieder dechiffriert werden.

Weitere Informationen erhalten Sie, wenn Sie den Mauszeiger über eine Einstellung bewegen.

Bitte konfigurieren Sie zu Ihrer Sicherheit die Verschlüsselung !

Betriebsart Aus
Zu Ihrer eigenen Sicherheit empfehlen wir Ihnen dringend, die Verschlüsselung nicht auszuschalten. Wählen Sie dagegen eine der unten aufge- führten Verschlüsselungsarten.

Bitte konfigurieren Sie zu Ihrer Sicherheit die Verschlüsselung !

Betriebsart WEP
WEP steht für Wired Equivalent Privacy. Im Modus WEP können Sie den benötigten Schlüssel in ASCII oder HEX Schreibweise eintragen. Es wird dann immer der gleiche Verschlüsselungs- algorithmus verwendet. Dies stellt heute keinen absoluten Schutz mehr dar.

Bitte konfigurieren Sie zu Ihrer Sicherheit die Verschlüsselung !

Betriebsart WPA2 mit Pre-shared key
In diesem Modus können sich nur WLAN Clients anmelden, welche die Verschlüsselung WPA2 (AES) unterstützen. Hier wird der Verschlüsselungs- algorithmus fortlaufend verändert. Dadurch wird eine höhere Sicherheit erreicht.


WPA2 funktioniert zur Zeit nur mit Windows® XP.

Bitte konfigurieren Sie zu Ihrer Sicherheit die Verschlüsselung !

Betriebsart WPA / WPA2 mit Pre-shared key
In diesem Modus können sich WLAN Clients anmelden, die entweder die Verschlüsselung WPA (TKIP) oder WPA2 (AES) unterstützen. Bei WPA / WPA2 wird der Verschlüsselungs- algorithmus fortlaufend verändert. Dadurch wird eine höhere Sicherheit erreicht.


WPA / WPA2 funktioniert zur Zeit nur mit Windows® XP.

Bitte konfigurieren Sie zu Ihrer Sicherheit die Verschlüsselung !

Betriebsart WPA2 mit RADIUS Server
RADIUS ist eine Abkürzung für Remote Authentication Dial-in User Service. Der Router leitet die Daten zur Authentifizierung und Autorisierung von Benutzern an einen externen Server weiter.

Diese Betriebsart wird hier nur aufgeführt, weil sie bereits konfiguriert ist. Die RADIUS-Konfiguration kann im Assistent nicht verändert werden.
