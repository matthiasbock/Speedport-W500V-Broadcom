Gesamtbudgetierung

Hier können Sie die Parameter für die Budgetierung festlegen.

Weitere Informationen erhalten Sie, wenn Sie den Mauszeiger über einen Menüpunkt bewegen.

Gesamtbudgetierung

Budgetierung
Hier schalten Sie die Budgetierung 'Ein' oder 'Aus'

Budgetierung

Vorgabe der Zeit
Wenn Sie die Vorgabe der Zeit aktivieren, muss der Haken im Kontrollkästchen erscheinen. Tragen Sie dann die Anzahl der Stunden ein, die Sie Online zulassen möchten. Sobald diese Zeit überschritten wird, erhält der Router keinen Zugang mehr ins Internet.

Budgetierung

Vorgabe der Datenmenge
Wenn Sie die Vorgabe der Datenmenge aktivieren, muss der Haken im Kontrollkästchen erscheinen. Tragen Sie dann die Datenmenge in MB ein, die Sie Online zulassen möchten. Sobald diese Datenmenge überschritten wird, erhält der Router keinen Zugang mehr ins Internet.

Budgetierung

Grenzwert zurücksetzen
Hier tragen sie den Tag ein, an dem die Grenzwerte für Zeit und Datenmenge wieder zurückgesetzt werden. Das Konto wird somit wieder aufgefüllt.

Bei Erreichen des Grenzwerts

Internetverbindung
Wenn Sie die Option 'Trennen' auswählen, wird eine bestehende Internetverbindung sofort getrennt wenn der Grenzwert der Zeit oder der Datenmenge erreicht ist.

Wenn Sie die Option 'Nicht Trennen'auswählen, wird eine bestehende Internetverbindung nicht sofort getrennt wenn der Grenzwert der Zeit oder der Datenmenge erreicht ist. Erst der nächste Versuch eine Internetverbindung aufzubauen wird verhindert.

Einstellung nicht änderbar

Um diese Einstellung ändern zu können, muss die Budgetierung eingeschaltet sein.
