Mit welchem Anbieter möchten Sie über das Internet telefonieren?

Mit diesem Router können Sie von einem angeschlossenen Telefon aus über das Internet telefonieren. Wählen Sie aus, ob Sie die Zugangsdaten für die DSL Telefonie von T-Online oder einem anderen Anbieter eintragen möchten.

Falls Sie die Zugangsdaten noch nicht haben, können Sie diese auch zu einem späteren Zeitpunkt eingeben. In diesem Fall werden eine oder mehrere Seiten des Assistenten übersprungen.

Weitere Informationen erhalten Sie, wenn Sie den Mauszeiger über eine Einstellung bewegen.

Zu späterem Zeitpunkt einrichten

Für die Einrichtung der Internet- telefonie benötigen Sie die Zugangsdaten für die Internet- rufnummer und einen funktionierenden Internetzugang.

Falls diese Voraussetzungen nicht erfüllt sind, können Sie die Internettelefonie erst zu einem späteren Zeitpunkt einrichten. In diesem Fall werden eine oder mehrere Seiten des Assistenten übersprungen.

Anbieter nicht verfügbar

Um T-Online auswählen zu können, muss ein funktionierender Internetzugang eingerichtet sein. Durchlaufen Sie hierzu den Assistenten und speichern Sie am Ende die Einstellungen. Nachdem Sie dann eine Internetverbindung erfolgreich aufgebaut haben können Sie den Assistenten noch mal durchlaufen und die Internettelefonie einrichten.
