Weitere Informationen erhalten Sie, wenn Sie den Mauszeiger über eine Einstellung bewegen.

Mitbenutzernummer / Suffix
Tragen Sie die Nummer des bei T-Online registrierten Mitbenutzers ein. Ist nur ein Mitbenutzer registriert, tragen Sie die Ziffernfolge '0001' ein. Die Mitbenutzernummer muss vierstellig eingeben werden.

Internet Zugangsdaten

Persönliches Kennwort
Tragen Sie hier Ihr persönliches Kennwort ein und wiederholen Sie die Eingabe im Feld darunter. Das Kennwort können Sie nur in der entsprechenden Anwendung von T-Online ändern.

Router-Einstellungen

Dauerverbindung
Eine Dauerverbindung ist in folgenden Fällen erforderlich:

  • Um die ständige Erreichbarkeit über Ihre DSL Telefonnummer zu gewährleisten.

  • Um Verzögerungen beim abgehenden Telefonieren zu vermeiden.

    • Wenn Sie die Dauerverbindung einschalten möchten, klicken Sie auf die Option 'Ein'.

    Mit eingeschalteter Dauerverbindung wird die Verbindung ins Internet immer aufrecht erhalten.


    Beachten Sie bitte, dass sich eine eingeschaltete Dauerverbindung bei einem zeitbasierenden Tarif auf die Kosten auswirkt.

    Router-Einstellungen

    Automatisch verbinden
    Hier können Sie auswählen, ob bei Aufruf Ihres Standard Browsers immer automatisch eine Internetverbindung herge- stellt wird. Möchten Sie dies nicht, so müssen Sie die Verbindung zum Internet jeweils manuell über die Startseite des Konfigurationsprogramms herstellen.


    Mit der Zusatzsoftware 'Online- Control' wird der Status der Internetverbindung in der Taskleiste mit einem Icon angezeigt.

    Per Mausklick auf das Icon kön- nen Sie unter anderem die Ver- bindung ins Internet aufbauen.

    OnlineControl finden Sie auf Ihrer Installations-CD und im Internet auf den T-Com Seiten.

    Router-Einstellungen

    Automatisch trennen
    Findet während der angegebe- nen Zeit kein Datenverkehr mit dem Internet statt, wird die Ver- bindung zum Internet automa- tisch getrennt.

    Router Einstellungen

    Zwangstrennung
    Die Dauerverbindung wird aus technischen Gründen einmal täglich getrennt. Hier können Sie die Zeitspanne einstellen, in der die Trennung erfolgen kann. Die Verbindung wird anschließend automatisch wieder aufgebaut.
