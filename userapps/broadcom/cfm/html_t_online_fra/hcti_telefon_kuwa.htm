Kurzwahlliste

Hier können Sie die Rufnummern eintragen die Sie dann per Kurzwahl anwählen können.

Kurzwahlliste

KW
Mit der Kurzwahl können Sie die Rufnummer anwählen, die in dem Feld Rufnummer eingetragen ist.

Bezeichnung
Damit Sie wissen um welche Rufnummer es sich handelt, vergeben Sie bitte eine Bezeichnung.

Rufnummer
Tragen Sie hier die Rufnummer ein, die Sie per Kurzwahl erreichen möchten.

Netze
Wählen Sie das Netz aus, über welches die Anwahl erfolgen soll. Wenn Sie beide auswählen, dann wird die Verbindung über eine freie Leitung versucht aufzubauen. Die Priorität der Leitungsbelegung können Sie unter dem Menüpunkt: 'Rufnummer gehend' festlegen.
