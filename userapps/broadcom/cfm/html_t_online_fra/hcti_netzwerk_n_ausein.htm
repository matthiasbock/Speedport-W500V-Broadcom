NAT

Hier können Sie NAT ein- oder ausschalten. Weiterhin können Sie festlegen, ob Sie einen Server verwenden möchten.

Weitere Informationen erhalten Sie, wenn Sie den Mauszeiger über einen Menüpunkt bewegen.

NAT

Betriebszustand
Die NAT-Funktion ist standardmäßig eingeschaltet.


Sie sollten Sie die NAT-Funktion nicht ausschalten, da dies die Funktion Ihres Netzwerks beeinträchigen kann. Das Gerät arbeitet sonst nur noch im Bridgebetrieb, d.h. es kann dann nur noch als DSL-Modem verwendet werden.

Standardserver

Server verwenden
Wenn Sie einen Server verwenden möchten, dann wählen Sie die Option 'Ein'.

Serverzuordnung

IP-Adresse
Zusätzlich zu Servern für bestimmte Dienste unterstützt NAT (Network Address Translation) einen Standard-Server. Der Standard-Server empfängt die Pakete von denjenigen Ports, die nicht definiert worden sind. Wenn Sie keine IP-Adresse für den Standard-Server festlegen, werden alle Pakete abgewiesen, die nicht explizit für definierte Ports bestimmt sind.

Tragen Sie hier die IP-Adresse des Servers ein, den Sie verwenden möchten.

Einstellung nicht änderbar

Um diese Einstellung ändern zu können, muss 'Server verwenden' eingeschaltet sein.

Einstellung nicht änderbar

Um diese Einstellung ändern zu können, muss der 'NAT Betriebszustand' eingeschaltet sein.
