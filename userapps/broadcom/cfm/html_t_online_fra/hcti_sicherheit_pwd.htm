Passwort Änderung

Damit die Router-Konfiguration vor unberechtigtem Zugriff geschützt ist, muss das im Auslieferungszustand eingetragene Passwort 0000 geändert werden.

Bewahren Sie das neue Passwort an einer sicheren Stelle vor Zugriff von Unbefugten auf.


Zu Ihrer Sicherheit wird das Konfigurationsprogramm automatisch beendet, nachdem es mehr als 10 Minuten nicht verwendet wurde.

Weitere Informationen erhalten Sie, wenn Sie den Mauszeiger über einen Menüpunkt bewegen.

Passwort Änderung

Altes Passwort
Tragen Sie hier Ihr bisheriges Passwort ein. Im Auslieferungszustand ist das Passwort 0000 eingetragen.

Passwort Änderung

Neues Passwort
Tragen Sie hier Ihr neues Passwort ein und wiederholen Sie die Eingabe im Feld darunter.

Das Passwort kann aus
4 bis 12 alphanumerischen Zeichen (A-Z, a-z, 0-9) bestehen, Groß-/Kleinschreibung wird berücksichtigt.

Passwort Änderung

Neues Passwort wiederholen
Wiederholen Sie die Eingabe des neuen Passwortes.

Das Passwort kann aus
4 bis 12 alphanumerischen Zeichen (A-Z, a-z, 0-9) bestehen, Groß-/Kleinschreibung wird berücksichtigt.
