Telefonie

Hier können Sie alle Einstellungen für den Bereich Telefonie vornehmen.

Weitere Informationen erhalten Sie, wenn Sie den Mauszeiger über einen Menüpunkt bewegen.

Telefonnummern

DSL Telefonie
Hier erreichen Sie die Eingabemaske zur Eingabe der Daten für die DSL Telefonie.

Telefonnummern

T-Net Anschluss
Hier können Sie festlegen, ob Sie einen T-Net Anschluss (Festnetzanschluss) am Router betreiben möchten.

Anschluss-Einstellungen
Hier können Sie für den
internen Anschluss die Leistungsmerkmale einstellen.

Leistungsmerkmale

Wahlregeln
Hier können Sie festlegen, welche Rufnummern über welche Anschlüssen angerufen werden.

Leistungsmerkmale

Notrufnummern
Hier können Sie Notrufnummern eintragen. Die Notrufnummern durchbrechen jede eventuell eingestellte Sperre.

Leistungsmerkmale

Sperrliste gehend
Hier können Sie Rufnummern eintragen, die gehend gesperrt werden sollen.

Leistungsmerkmale

Sperrliste kommend
Hier können Sie Rufnummern eintragen, die kommend gesperrt werden sollen. Anrufer erhalten dann immer besetzt.

Kurzwahlliste
Hier können Sie Rufnummern eintragen, die über eine Kurzwahl angewählt werden können.
