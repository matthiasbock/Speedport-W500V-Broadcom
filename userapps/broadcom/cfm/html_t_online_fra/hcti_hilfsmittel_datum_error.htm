Datum & Uhrzeit

Geben Sie einen Zeitserver an, von dem das aktuelle Datum und die Uhrzeit übernommen werden kann.

In der Werkseinstellung ist der
T-Online Zeitserver mit folgender Bezeichnung eingetragen:
 ntp1.sda.t-online.de

Ändern Sie dies nur, wenn Sie sicher sind, dass sie einen anderen funktionierenden Zeitserver angeben können.

Datum und Uhrzeit werden automatisch von diesem Zeit- server im Internet bezogen.

Dies geschieht in folgenden Fällen:

  • Wenn der Router einge- schaltet wird.
  • Täglich einmal, aber nur wenn der Router Online ist.
